Rating <ui-input-rating> is an interactive star rating control. Click a star to set the value, click the same star again to clear it. Hover feedback and keyboard navigation are built in. Basic Usage Set value for the initial rating and max for the total number of stars. The default is 5 stars.
<ui-input-rating label="5 stars" value="3" max="5"></ui-input-rating> <ui-input-rating label="10 stars" value="7" max="10"></ui-input-rating>
Help & Error Use help for guidance text and error for validation messages. When error is set, the help text is hidden.
<ui-input-rating label="With help" help="How would you rate this?" value="4"></ui-input-rating> <ui-input-rating label="With error" error="A rating is required" required></ui-input-rating>
Star Colours Use the color attribute with a palette token or any CSS colour to change the active star colour. Default is amber-400.
<ui-input-rating label="Amber" value="3" color="amber-400"></ui-input-rating> <ui-input-rating label="Rose" value="3" color="rose-400"></ui-input-rating> <ui-input-rating label="Indigo" value="3" color="indigo-400"></ui-input-rating> <ui-input-rating label="Emerald" value="3" color="emerald-400"></ui-input-rating>
Sizes Use size with CSS named sizes or any CSS length. Stars scale proportionally via em.
<ui-input-rating label="Small" size="small" value="3"></ui-input-rating> <ui-input-rating label="Default" value="3"></ui-input-rating> <ui-input-rating label="Large" size="large" value="3"></ui-input-rating> <ui-input-rating label="X-Large" size="x-large" value="3"></ui-input-rating>
Read Only The readonly attribute prevents changes but keeps the component visually active (no dimming). Useful for displaying a submitted rating.
<ui-input-rating label="Read only" value="4" readonly></ui-input-rating>
Disabled The disabled attribute dims the component and prevents all interaction.
<ui-input-rating label="Disabled" value="2" disabled></ui-input-rating>
play-circle Playground Interactively configure every property.
