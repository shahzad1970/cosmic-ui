Slider <ui-input-slider> provides a range slider with optional value display, label, help/error text, and accent colour. Basic Usage Set min, max, step, and value. Add showvalue to display the current number.
<ui-input-slider label="Volume" min="0" max="100" value="35" showvalue></ui-input-slider>
Help & Error Use help for guidance and error for validation messages. When required is set, an asterisk appears next to the label.
<ui-input-slider label="Brightness" help="Drag to adjust" required showvalue> </ui-input-slider> <ui-input-slider label="Contrast" error="Contrast must be above 30" showvalue> </ui-input-slider>
Disabled Add the disabled attribute to prevent interaction.
<ui-input-slider label="Locked" value="60" showvalue disabled></ui-input-slider>
Sizes The size attribute accepts CSS named sizes (x-small … xx-large) or any CSS length.
<ui-input-slider label="Small" size="small" showvalue></ui-input-slider> <ui-input-slider label="Default" showvalue></ui-input-slider> <ui-input-slider label="Large" size="large" showvalue></ui-input-slider>
Background (Accent) Use palette tokens or any CSS colour for the slider thumb and focus ring.
<ui-input-slider background="emerald-500" showvalue></ui-input-slider> <ui-input-slider background="rose-500" showvalue></ui-input-slider>
