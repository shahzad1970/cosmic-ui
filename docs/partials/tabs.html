Tabs <ui-tabs> manages tab selection and panels using <ui-tab> children. Basic Usage Each ui-tab provides the label and panel content.
Overview content goes here. Details panel content. Disabled tab content.
<ui-tabs> <ui-tab label="Overview">...</ui-tab> <ui-tab label="Details">...</ui-tab> <ui-tab label="Activity" disabled>...</ui-tab> </ui-tabs>
Placement Place the tab bar on any edge of the container.
Alpha panel Beta panel Gamma panel
<ui-tabs placement="start">...</ui-tabs>
Icon Labels Use short labels with icon-like text and add icons inside panels if needed.
home Home panel content. settings Settings panel content. credit-card Billing panel content.
<ui-tab label="Home *">...</ui-tab> <ui-tab label="Settings *">...</ui-tab>
Sizes Scale the entire tab component with the size attribute.
x-small tabs content. Second panel. Default size tabs content. Second panel. Large tabs content. Second panel.
<ui-tabs size="x-small">...</ui-tabs> <ui-tabs>...</ui-tabs> <ui-tabs size="large">...</ui-tabs>
Controls
First panel content. Second panel content. Third panel content.
