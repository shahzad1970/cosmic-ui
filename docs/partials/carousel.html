Carousel <ui-carousel> creates a slide-based content carousel with prev/next arrows, dot indicators, optional auto-play with pause-on-hover, looping, keyboard navigation, and touch-swipe support. Basic Carousel Place <ui-carousel-slide> children inside the carousel. Each slide can set its own background and color.
Slide One Slide Two Slide Three
<ui-carousel loop> <ui-carousel-slide background="indigo-600">Slide One</ui-carousel-slide> <ui-carousel-slide background="emerald-600">Slide Two</ui-carousel-slide> <ui-carousel-slide background="rose-600">Slide Three</ui-carousel-slide> </ui-carousel>
Autoplay Set autoplay to an interval in milliseconds. The carousel pauses on hover and resumes on mouse-leave.

<ui-carousel loop autoplay="3000"> <ui-carousel-slide background="sky-600">...</ui-carousel-slide> <ui-carousel-slide background="violet-600">...</ui-carousel-slide> <ui-carousel-slide background="amber-600">...</ui-carousel-slide> </ui-carousel>
No Controls Set navigation and indicators to false via JavaScript to hide arrows and dots. Users can still swipe or use keyboard arrows.
Minimal — Slide 1 Minimal — Slide 2 Minimal — Slide 3
const el = document.querySelector('ui-carousel'); el.navigation = false; el.indicators = false;
Rich Content Slides accept any HTML content — cards, images, forms, or nested components.

<ui-carousel loop> <ui-carousel-slide background="indigo-50" color="indigo-900"> <ui-avatar background="indigo-600" size="x-large">JS</ui-avatar> <ui-text type="heading">Jane Smith</ui-text> </ui-carousel-slide> ... </ui-carousel>
Background Images Set the image attribute on a slide to use a photo background. Combine with background as a fallback colour. Text defaults to white on image slides.
Mountain Lake Misty Forest Coastal Road
<ui-carousel loop> <ui-carousel-slide image="https://picsum.photos/id/1015/800/400"> Mountain Lake </ui-carousel-slide> <ui-carousel-slide image="https://picsum.photos/id/1018/800/400"> Misty Forest </ui-carousel-slide> </ui-carousel>
Elevation Add depth with elevation (1–5). The carousel gains a box-shadow, making it pop off the page.
Elevation 2 Slide Two Elevation 5 Slide Two
<ui-carousel loop elevation="2">...</ui-carousel> <ui-carousel loop elevation="5">...</ui-carousel>
Sizes Scale the entire carousel with the size attribute. All inner elements scale proportionally.
Small Slide A Slide B Large Slide A Slide B
<ui-carousel loop size="small">...</ui-carousel> <ui-carousel loop size="large">...</ui-carousel>

API Reference ui-carousel
AttributeTypeDefaultDescription
autoplayNumber0Auto-advance interval in milliseconds. 0 disables autoplay. Pauses on hover.
loopBooleanfalseWrap around: when reaching the last slide, next() goes back to the first (and vice-versa)
navigationBooleantrueShow previous/next arrow buttons overlaid on the carousel
indicatorsBooleantrueShow dot indicators at the bottom of the carousel
backgroundString—Carousel container background colour (palette token or CSS colour)
colorString—Carousel container text colour (palette token or CSS colour)
sizeString—Scale the entire carousel — named keyword (xx-small … xxx-large) or CSS length
elevationString—Shadow depth (1–5) or raw CSS box-shadow value

ui-carousel-slide
AttributeTypeDefaultDescription
backgroundString—Slide background colour (palette token or CSS colour). Auto-sets contrasting text colour.
colorString—Slide text colour — overrides auto-contrast when set explicitly
imageString—Background image URL. Covers the slide with background-size: cover. Text defaults to white.
