Pagination <ui-pagination> renders page navigation with smart ellipsis truncation, previous/next arrows, and an optional compact mode. Basic Usage Set total for the number of pages and current for the active page.
<ui-pagination total="20" current="5" background="indigo-500"></ui-pagination> <ui-pagination total="20" current="5" background="emerald-500" compact></ui-pagination>
Sizes Scale pagination with the size attribute.
<ui-pagination total="10" current="3" size="x-small"></ui-pagination> <ui-pagination total="10" current="3"></ui-pagination> <ui-pagination total="10" current="3" size="large"></ui-pagination>
Siblings Control how many page numbers appear on each side of the current page.
<ui-pagination total="30" current="15" siblings="0"></ui-pagination> <ui-pagination total="30" current="15" siblings="1"></ui-pagination> <ui-pagination total="30" current="15" siblings="3"></ui-pagination>
Disabled Add disabled to prevent all interaction.
<ui-pagination total="10" current="4" background="indigo-500" disabled></ui-pagination>
play-circle Playground Adjust pagination settings interactively.
