Spacer <ui-layout-spacer> is a flexible space filler. Without a size it grows to fill available space; with size it becomes a fixed gap. Diagram Flexible spacer pushes content apart.
Flexible Spacer Use in a horizontal stack to push content to the edges.

Fixed Gap Set size to create a fixed vertical or horizontal gap.
