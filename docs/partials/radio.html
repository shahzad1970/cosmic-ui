Radio <ui-input-radio-group> manages single-choice options with <ui-input-radio> children. Supports help text, error state, sizes, and accent colours. Basic Usage Wrap <ui-input-radio> elements inside a group. Set value on the group to pre-select an option.
<ui-input-radio-group label="Plan" value="pro"> <ui-input-radio value="starter" label="Starter"></ui-input-radio> <ui-input-radio value="pro" label="Pro"></ui-input-radio> <ui-input-radio value="team" label="Team"></ui-input-radio> </ui-input-radio-group>
Help & Error Messages Use help for guidance text below the group. Set error to show a red error message (replaces help text).
<ui-input-radio-group label="Frequency" help="Choose how often you'd like updates" value="weekly"> <ui-input-radio value="daily" label="Daily"></ui-input-radio> <ui-input-radio value="weekly" label="Weekly"></ui-input-radio> </ui-input-radio-group> <ui-input-radio-group label="Priority" error="Please select a priority" required> <ui-input-radio value="low" label="Low"></ui-input-radio> <ui-input-radio value="high" label="High"></ui-input-radio> </ui-input-radio-group>
Disabled Add disabled to the group to disable all radios, or on individual <ui-input-radio> elements selectively.
<ui-input-radio-group label="Group disabled" disabled value="a"> <ui-input-radio value="a" label="Option A"></ui-input-radio> <ui-input-radio value="b" label="Option B"></ui-input-radio> </ui-input-radio-group> <ui-input-radio-group label="Single option disabled" value="yes"> <ui-input-radio value="yes" label="Yes"></ui-input-radio> <ui-input-radio value="no" label="No" disabled></ui-input-radio> </ui-input-radio-group>
Sizes Use size on the group with CSS keywords (x-small … xx-large) or any CSS length.
<ui-input-radio-group label="Small" size="small" value="a"> <ui-input-radio value="a" label="Alpha"></ui-input-radio> </ui-input-radio-group> <ui-input-radio-group label="Large" size="large" value="a"> <ui-input-radio value="a" label="Alpha"></ui-input-radio> </ui-input-radio-group>
Background & Color Set background on the group to a palette token (e.g. emerald-600) or any CSS colour to change the radio accent.
<ui-input-radio-group label="Emerald" background="emerald-600" value="a"> <ui-input-radio value="a" label="A"></ui-input-radio> <ui-input-radio value="b" label="B"></ui-input-radio> </ui-input-radio-group>
play-circle Playground Switch selected value, size, accent colour, and error text.
