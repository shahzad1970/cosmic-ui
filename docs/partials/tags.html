Tags <ui-input-tags> is a free-form tag / chip input. Type text and press Enter or comma to add a tag, click × to remove. Duplicates are automatically prevented. Basic Usage Pre-populate with a comma-separated value. Press Backspace on an empty input to remove the last tag.
<ui-input-tags label="Skills" value="JavaScript, CSS, HTML"></ui-input-tags>
Help & Error Use help for guidance and error for validation. Error state turns the wrapper border red.
<ui-input-tags label="With help" help="Add up to 5 tags" max="5"></ui-input-tags> <ui-input-tags label="With error" error="At least one tag is required"></ui-input-tags>
Max Tags Set max to limit the number of tags. When the limit is reached, the input field is hidden.
<ui-input-tags label="Top 3 colours" max="3" value="Red, Blue"></ui-input-tags>
Tag Colours Use the color attribute with a palette token to change the tag badge colour.
<ui-input-tags label="Indigo" color="indigo-100" value="Tag A, Tag B"></ui-input-tags> <ui-input-tags label="Emerald" color="emerald-100" value="Tag A, Tag B"></ui-input-tags> <ui-input-tags label="Rose" color="rose-100" value="Tag A, Tag B"></ui-input-tags>
Sizes Use size with CSS named sizes or any CSS length.
<ui-input-tags label="Small" size="small" value="A, B"></ui-input-tags> <ui-input-tags label="Default" value="A, B"></ui-input-tags> <ui-input-tags label="Large" size="large" value="A, B"></ui-input-tags>
Disabled
<ui-input-tags label="Disabled" value="Locked, Tag" disabled></ui-input-tags>
