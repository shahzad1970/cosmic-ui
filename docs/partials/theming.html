Theming Cosmic UI is fully themeable through CSS custom properties. Every design token uses the --ui- prefix and can be overridden at any level — globally on :root, scoped to a container, or on individual components. How It Works On first import, Cosmic UI injects a :root stylesheet with all design tokens. Components reference these tokens internally. Override any --ui-* variable to change the look-and-feel — no build step, no config files.
<!-- Cosmic UI injects tokens automatically --> <script type="module" src="path/to/cosmic-ui/src/index.js"></script> <!-- Override any token in your own stylesheet --> <style> :root { --ui-focus-ring: var(--ui-emerald-500); --ui-border-color: var(--ui-gray-200); --ui-radius: 0.5em; } </style>

You can also reference tokens directly in CSS using var(--ui-*):
.my-custom-card { background: var(--ui-indigo-50); border: 1px solid var(--ui-indigo-200); color: var(--ui-indigo-900); }
Shared UI Tokens Beyond the colour palette, Cosmic UI defines shared design tokens that control typography, focus rings, borders, and more.
Token Default Description
--ui-font System font stack Base font family for all components
--ui-font-mono Monospace stack Font for code, OTP, and monospaced content
--ui-focus-ring indigo-500 Outline colour for focused elements
--ui-border-color gray-300 Default border colour for inputs, cards, etc.
--ui-text-color gray-900 Primary text colour
--ui-text-muted gray-500 Secondary / muted text colour
--ui-bg #fff Page background
--ui-bg-subtle gray-50 Subtle background for alternating rows, etc.
--ui-radius 0.375em Default border-radius
--ui-button-radius var(--ui-radius) Border-radius specifically for buttons
Overriding Tokens Override tokens at any level — globally, per section, or per component. Global Override Change the entire app's look by overriding :root tokens.
<style> :root { /* Rounded buttons */ --ui-button-radius: 9999px; /* Custom focus ring */ --ui-focus-ring: var(--ui-emerald-400); /* Softer borders */ --ui-border-color: var(--ui-gray-200); /* Custom font */ --ui-font: "Inter", sans-serif; } </style>
Scoped Override Apply tokens to a specific section without affecting the rest of the page.
Pill Button Rounded
<div style="--ui-button-radius: 9999px;"> <ui-button background="indigo-600">Pill Button</ui-button> <ui-button background="emerald-500">Rounded</ui-button> </div>
Sizing System All component dimensions use em units. Change a component's size attribute to scale everything proportionally — padding, border-radius, icons, and text all adjust together.

Dark Mode Cosmic UI doesn't ship a built-in dark mode toggle, but it's trivial to build one by swapping token values.
<style> @media (prefers-color-scheme: dark) { :root { --ui-bg: var(--ui-gray-900); --ui-bg-subtle: var(--ui-gray-800); --ui-text-color: var(--ui-gray-100); --ui-text-muted: var(--ui-gray-400); --ui-border-color: var(--ui-gray-700); } } /* Or use a class toggle */ .dark { --ui-bg: var(--ui-gray-900); --ui-bg-subtle: var(--ui-gray-800); --ui-text-color: var(--ui-gray-100); --ui-text-muted: var(--ui-gray-400); --ui-border-color: var(--ui-gray-700); } </style>
Quick Reference Common theming recipes at a glance. Change brand colour: Override --ui-indigo-* shades with your brand's primary hue, or use a different palette family on components. Rounded everything: Set --ui-radius: 9999px on :root for fully-rounded corners on all components. Custom font: Set --ui-font: "YourFont", sans-serif on :root. All components inherit from <ui-app>. Scale globally: Change font-size on <ui-app> and all em-based components scale proportionally.
