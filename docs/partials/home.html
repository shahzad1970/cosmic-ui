Cosmic UI A pure-JavaScript web component framework. Zero dependencies. Shadow DOM. No build step.
Get Started Browse Components
Why Cosmic UI? Everything you need to build modern interfaces — in one tiny, dependency-free library.
box Zero Dependencies Pure vanilla JavaScript and standard Web APIs. No framework lock-in, no bundler, no transpiler. Just import and go. layers Shadow DOM Every component is fully encapsulated. Styles don't leak in or out. Safe to use alongside any existing CSS or framework. zap No Build Step Ship ES modules directly. Works with a simple <script type="module"> tag. Perfect for prototyping and production alike. palette Themeable Full colour palette with 19 hues × 11 shades. Override any --ui-* token to match your brand in seconds. accessibility Accessible Semantic roles, keyboard support, and focus management are baked in. Components use native elements internally for best-in-class a11y. maximize Responsive The app shell collapses automatically on small screens. All sizing uses em units — scale by changing font-size.
Quick Start Add a single <script> tag and start building. <!DOCTYPE html> <html lang="en"> <head> <script type="module" src="path/to/cosmic-ui/src/index.js"></script> </head> <body> <ui-app> <ui-app-banner background="indigo-700">My App</ui-app-banner> <ui-app-header>Dashboard</ui-app-header> <ui-app-main> <ui-card elevation="2"> <ui-card-body>Hello, Cosmic UI!</ui-card-body> </ui-card> </ui-app-main> <ui-app-footer>© 2025</ui-app-footer> </ui-app> </body> </html> Components 42+ components covering layout, forms, navigation, feedback, and more.
