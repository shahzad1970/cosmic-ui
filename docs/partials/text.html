Text <ui-text> is a universal text container that supports semantic headings, paragraphs, and rich styling controls. Basic Usage Use type for semantic headings and paragraphs.
Section Title This is a paragraph with default spacing. Muted supporting text.
<ui-text type="heading" size="large">Section Title</ui-text> <ui-text type="paragraph">This is a paragraph...</ui-text> <ui-text muted>Muted supporting text.</ui-text>
Heading Sizes Headings render an <h2> inside Shadow DOM. Control the visual size with the size attribute.
Display Page Title Section Subsection Default (1.5em) Small heading
<ui-text type="heading" size="xxx-large">Display</ui-text> <ui-text type="heading" size="xx-large">Page Title</ui-text> <ui-text type="heading" size="x-large">Section</ui-text> <ui-text type="heading" size="large">Subsection</ui-text> <ui-text type="heading">Default (1.5em)</ui-text> <ui-text type="heading" size="medium">Small heading</ui-text>
Colours & Weight Palette tokens for color and named or numeric weight.
Semibold indigo Bold emerald Light rose Thin gray Bold shorthand Muted text
<ui-text color="indigo-600" weight="semibold">Semibold indigo</ui-text> <ui-text color="emerald-600" weight="bold">Bold emerald</ui-text> <ui-text color="rose-500" weight="light">Light rose</ui-text> <ui-text bold>Bold shorthand</ui-text> <ui-text muted>Muted text</ui-text>
Typography Fine-tune leading, letter-spacing, indent, and transform.
Section Label This paragraph has loose line-height for extra breathing room between lines of text that wrap to multiple lines. Tight leading on large bold text creates a dense, impactful headline style. This paragraph has a 2em first-line indent, similar to traditional typographic book paragraphs.
<ui-text block size="small" transform="uppercase" spacing="widest" weight="semibold" color="gray-500">Section Label</ui-text> <ui-text block leading="loose">Loose line-height...</ui-text> <ui-text block leading="tight" size="large" weight="bold">Tight leading...</ui-text> <ui-text block indent="2em">First-line indent...</ui-text>
Decoration & Style Boolean shorthands and the decoration attribute for text styling.
Italic text Underlined text Strikethrough (deprecated) Wavy underline const x = monospace();
<ui-text italic>Italic text</ui-text> <ui-text underline>Underlined text</ui-text> <ui-text strike muted>Strikethrough (deprecated)</ui-text> <ui-text decoration="underline wavy" color="rose-500">Wavy underline</ui-text> <ui-text mono size="small">const x = monospace();</ui-text>
Truncation & Line Clamp Use truncate for single-line ellipsis, or lines for multi-line clamping.
truncate This is a very long sentence that will be truncated with an ellipsis when it overflows the container width. lines="2" This paragraph is clamped to two lines. Any text beyond the second line will be hidden and an ellipsis will appear at the break point. Lorem ipsum dolor sit amet. lines="3" This paragraph is clamped to three lines. Any text beyond the third line will be hidden and an ellipsis will appear. Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt.
<ui-text truncate>Long text truncated with ellipsis...</ui-text> <ui-text lines="2">Two-line clamp with ellipsis...</ui-text> <ui-text lines="3">Three-line clamp...</ui-text>
Alignment Control text alignment with the align attribute.
Left aligned (default) Center aligned Right aligned Justified text spreads evenly across the full width of the line, adding extra space between words.
<ui-text block align="left">Left aligned</ui-text> <ui-text block align="center">Center aligned</ui-text> <ui-text block align="right">Right aligned</ui-text> <ui-text block align="justify">Justified text...</ui-text>

API Reference
AttributeTypeDefaultDescription
typeString—Semantic type: heading or paragraph
sizeString—Font size (named keyword or CSS length)
colorString—Text colour (palette token or CSS)
weightString—Font weight: thin–black or 100–900
alignString—Text alignment: left, center, right, justify
transformString—Text transform: uppercase, lowercase, capitalize
decorationString—Text decoration: underline, line-through, overline, or full CSS shorthand
leadingString—Line height: none, tight, snug, normal, relaxed, loose or CSS value
spacingString—Letter spacing: tighter–widest or CSS value
wrapString—Text wrap: balance, pretty, nowrap, wrap or CSS value
breakString—Word break: all, words, keep or CSS value
indentString—First-line text indentation (CSS length)
marginString—Margin shorthand (headings default 0 0 0.5em, paragraphs 0 0 1em)
paddingString—Padding shorthand
fontString—Custom font family override
