Spinner <ui-spinner> renders a circular loading indicator with configurable colour, thickness, speed, and an optional overlay mode. Basic Usage Drop a spinner inline to indicate loading state.
<ui-spinner></ui-spinner> <ui-spinner color="indigo-500"></ui-spinner>
Sizes Scale with size keywords or any CSS length.
<ui-spinner size="x-small"></ui-spinner> <ui-spinner size="small"></ui-spinner> <ui-spinner></ui-spinner> <!-- default --> <ui-spinner size="large"></ui-spinner> <ui-spinner size="x-large"></ui-spinner>
Overlay Use overlay to fill the nearest positioned parent with a translucent backdrop and centred spinner.
<div style="position:relative;height:6em"> <ui-spinner overlay color="indigo-500"></ui-spinner> </div>
play-circle Playground Adjust size, colour, thickness, speed, and overlay mode.
