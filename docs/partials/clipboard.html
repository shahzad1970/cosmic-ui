Clipboard <ui-clipboard> copies text to the clipboard with a single click and shows brief visual feedback. Basic Usage Set the value attribute to the text that should be copied.
Copy command Copy text Flat style Outline style
<ui-clipboard value="npm install ui-components">Copy command</ui-clipboard> <ui-clipboard value="Hello World!" background="indigo-500">Copy text</ui-clipboard> <ui-clipboard value="secret-token" flat>Flat style</ui-clipboard> <ui-clipboard value="outlined" outline>Outline style</ui-clipboard>
Custom Feedback Change the confirmation text with feedback and the display duration with duration (ms).
Copy token Copy email
<ui-clipboard value="TOKEN-123" feedback="Token copied!" duration="3000"> Copy token </ui-clipboard> <ui-clipboard value="my@email.com" feedback="Email saved!" duration="1000"> Copy email </ui-clipboard>
Sizes Use the size attribute to control button size.
X-Small Small Default Large X-Large
<ui-clipboard value="xs" size="x-small">X-Small</ui-clipboard> <ui-clipboard value="md">Default</ui-clipboard> <ui-clipboard value="lg" size="large">Large</ui-clipboard>
Disabled Add the disabled attribute to prevent copying.
Disabled
<ui-clipboard value="can't copy this" disabled>Disabled</ui-clipboard>
Listening for Copy Listen for ui-copy to react when text has been copied.
const clip = document.querySelector('ui-clipboard'); clip.addEventListener('ui-copy', (e) => { console.log('Copied:', e.detail.value); });
