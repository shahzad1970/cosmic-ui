File <ui-input-file> is a drag-and-drop file input with a drop zone UI, file list, and size formatting. Basic Usage Set accept to filter file types. Drag files onto the drop zone or click to browse.
<ui-input-file label="Upload files" accept=".png,.jpg,image/*"></ui-input-file>
Help & Error Provide guidance with help and validation feedback with error. Add required for the asterisk indicator.
<ui-input-file label="Resume" help="PDF or DOCX, max 5 MB" accept=".pdf,.docx" required> </ui-input-file> <ui-input-file label="Photo" error="File is required" accept="image/*"> </ui-input-file>
Multiple Files Set multiple to allow selecting more than one file.
<ui-input-file label="Attachments" multiple accept="image/*,.pdf"></ui-input-file>
Disabled Add disabled to prevent file selection.
<ui-input-file label="Locked" disabled></ui-input-file>
Sizes The size attribute accepts CSS named sizes or any CSS length.
<ui-input-file label="Small" size="small"></ui-input-file> <ui-input-file label="Default"></ui-input-file> <ui-input-file label="Large" size="large"></ui-input-file>
Background (Accent) Use a palette token or CSS colour for the drop-zone accent.
<ui-input-file label="Indigo" background="indigo-500"></ui-input-file> <ui-input-file label="Teal" background="teal-500"></ui-input-file>
