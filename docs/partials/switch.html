Switch <ui-input-switch> is a toggle switch for on/off states with optional label, help text, error state, and full colour support. Basic Usage Use checked to show the enabled state.
<ui-input-switch label="Notifications" checked></ui-input-switch> <ui-input-switch label="Dark mode"></ui-input-switch> <ui-input-switch label="Airplane mode" disabled></ui-input-switch>
Help & Error Messages Use help for guidance text below the label. Set error to show a red error message (replaces help text).
<ui-input-switch label="Email digest" help="Receive a weekly summary" checked></ui-input-switch> <ui-input-switch label="Terms of service" error="You must accept to continue" required></ui-input-switch>
Disabled Add disabled to prevent interaction.
<ui-input-switch label="Disabled off" disabled></ui-input-switch> <ui-input-switch label="Disabled on" disabled checked></ui-input-switch>
Sizes Use size with CSS keywords (x-small … xx-large) or any CSS length.
<ui-input-switch label="Extra small" size="x-small" checked></ui-input-switch> <ui-input-switch label="Large" size="large" checked></ui-input-switch> <ui-input-switch label="Extra large" size="x-large" checked></ui-input-switch>
Background & Color Set background to a palette token (e.g. emerald-600) or any CSS colour to change the track accent.
<ui-input-switch label="Emerald" background="emerald-600" checked></ui-input-switch> <ui-input-switch label="Rose" background="rose-600" checked></ui-input-switch>
play-circle Playground Toggle states, size, accent colour, and error text.
