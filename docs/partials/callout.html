Callout <ui-callout> highlights important messages, tips, warnings, and alerts with colour, elevation, and dismiss controls. Basic Usage Use a light background with a matching dark color for semantic callouts.
info This is an informational message. alert-triangle Warning — this action cannot be undone. x-circle Something went wrong. Please try again. check-circle Changes saved successfully.
<ui-callout background="sky-50" color="sky-700"> <ui-icon size="small">info</ui-icon> Informational message. </ui-callout> <ui-callout background="amber-50" color="amber-700"> <ui-icon size="small">alert-triangle</ui-icon> Warning. </ui-callout> <ui-callout background="rose-50" color="rose-700"> <ui-icon size="small">x-circle</ui-icon> Error. </ui-callout> <ui-callout background="emerald-50" color="emerald-700"> <ui-icon size="small">check-circle</ui-icon> Success. </ui-callout>
Bold Colours Use saturated backgrounds — text colour auto-contrasts when color is omitted.
Indigo callout with auto-contrast white text. Emerald callout — auto white text. Rose callout — auto white text. Dark callout — auto white text.
<ui-callout background="indigo-500">Auto-contrast text.</ui-callout> <ui-callout background="emerald-600">Green callout.</ui-callout> <ui-callout background="rose-500">Rose callout.</ui-callout> <ui-callout background="gray-900">Dark callout.</ui-callout>
Dismissible Add dismissible for a close button. The callout fires ui-dismiss and removes itself.
info You can dismiss this callout. alert-triangle Dismissible warning.
<ui-callout background="sky-50" color="sky-700" dismissible> Dismiss me with the ✕ button. </ui-callout>
Elevation Add elevation (1–5) for shadow depth with a subtle border.
Elevation 1 — subtle shadow. Elevation 3 — medium shadow. Elevation 5 — deep shadow.
<ui-callout background="sky-50" elevation="1">Subtle</ui-callout> <ui-callout background="sky-50" elevation="3">Medium</ui-callout> <ui-callout background="sky-50" elevation="5">Deep</ui-callout>
Sizes Scale with named keywords or any CSS length.
Extra-small callout Small callout Default size callout Large callout
<ui-callout size="x-small">Extra-small</ui-callout> <ui-callout size="small">Small</ui-callout> <ui-callout>Default size</ui-callout> <ui-callout size="large">Large</ui-callout>
Rich Content Callouts accept any slotted content — headings, lists, buttons, icons.

API Reference
AttributeTypeDefaultDescription
backgroundString—Background colour (palette token or CSS colour). Text auto-contrasts if color is omitted.
colorStringautoText colour (auto-contrasts against background if omitted)
sizeString—Callout font-size (named keyword or CSS length)
elevationString—Shadow depth: 1–5 (adds border + shadow) or raw CSS box-shadow
dismissibleBooleanfalseShow a close button; fires ui-dismiss and removes element on click
Events
EventDetailDescription
ui-dismiss—Fired when the dismiss button is clicked (requires dismissible). The callout removes itself from the DOM.
