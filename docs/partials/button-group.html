Button Group <ui-button-group> groups buttons into a connected strip with merged borders and radii. Basic Usage Wrap <ui-button> elements to create a seamless toolbar.
Left Center Right
<ui-button-group> <ui-button background="indigo-600">Left</ui-button> <ui-button background="indigo-600">Center</ui-button> <ui-button background="indigo-600">Right</ui-button> </ui-button-group>
Vertical Add the vertical attribute to stack buttons top-to-bottom.
Top Middle Bottom
<ui-button-group vertical> <ui-button background="gray-200">Top</ui-button> <ui-button background="gray-200">Middle</ui-button> <ui-button background="gray-200">Bottom</ui-button> </ui-button-group>
Outline & Flat Variants Set variants on the child buttons — the group handles border merging automatically.
Left Center Right Left Center Right
<ui-button-group> <ui-button background="indigo-600" outline>Left</ui-button> <ui-button background="indigo-600" outline>Center</ui-button> <ui-button background="indigo-600" outline>Right</ui-button> </ui-button-group> <ui-button-group> <ui-button background="indigo-600" flat>Left</ui-button> <ui-button background="indigo-600" flat>Center</ui-button> <ui-button background="indigo-600" flat>Right</ui-button> </ui-button-group>
Sizes Set the size attribute on the group to apply a uniform size to all child buttons.
A B C A B C A B C
<ui-button-group size="x-small">…</ui-button-group> <ui-button-group>…</ui-button-group> <ui-button-group size="large">…</ui-button-group>
