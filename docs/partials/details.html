Details An expandable disclosure widget. Click the summary to reveal or hide content. Built on native <details> / <summary> for full accessibility and keyboard support. Import <script type="module" src="path/to/cosmic-ui/src/index.js"></script> Basic Usage Set the summary attribute for the clickable heading. Place content inside the default slot.
Cosmic UI is a pure-JavaScript web component library with zero dependencies. It uses Shadow DOM, adopted stylesheets, and vanilla JS — no build step required.

API Reference Attributes
Attribute Type Default Description
summary String "Details" Text shown in the clickable header
open Boolean false Whether the content is expanded
background String — Background colour (palette token or CSS colour)
color String — Text colour (palette token or CSS colour)
size String — Font-size keyword or CSS length
elevation String — Shadow depth 1–5, or raw CSS box-shadow
outline Boolean false Border style with no shadow
flat Boolean false No border or shadow
Events
Event Detail Description
ui-toggle { open: Boolean } Fired when the open state changes

Slot Description
(default) The expandable content revealed when opened
