Stack <ui-layout-stack> is a flexbox-based layout container for vertical or horizontal stacking with consistent spacing. Diagram Vertical (default) and horizontal stacking.
Basic Stack The default direction is vertical. Use gap to control spacing.
First Second Third
<ui-layout-stack gap="0.5em"> <ui-card>...</ui-card> <ui-card>...</ui-card> <ui-card>...</ui-card> </ui-layout-stack>
Horizontal Alignment Use horizontal, align, and justify to control alignment.
Left Center Right
<ui-layout-stack horizontal gap="0.5em" align="center" justify="between"> <ui-badge>Left</ui-badge> <ui-badge>Center</ui-badge> <ui-badge>Right</ui-badge> </ui-layout-stack>
Wrapping & Padding Use wrap to allow items to flow to the next line and padding for inner spacing.
Tag A Tag B Tag C Tag D Tag E
<ui-layout-stack horizontal wrap gap="0.4em" padding="0.5em"> <ui-badge>Tag A</ui-badge> <ui-badge>Tag B</ui-badge> <ui-badge>Tag C</ui-badge> </ui-layout-stack>
play-circle Playground Toggle direction and spacing to see the stack update in real time.
