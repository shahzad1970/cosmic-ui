Container <ui-layout-container> centers content and constrains it to a max width with consistent horizontal padding. Diagram Centered content with max-width and padding.
Default Container Defaults to a 1200px max width with 1.5em horizontal padding.
Centered Content This content is constrained by the container.
<ui-layout-container padding="1em"> <ui-text type="heading">Centered Content</ui-text> <ui-text type="paragraph">This content is constrained by the container.</ui-text> </ui-layout-container>
Size Presets Use size to pick a preset width: small (640px), medium (960px), large (1200px), x-large (1400px), full (100%).
Small container (640px max)
<ui-layout-container size="small" padding="1em"> <ui-text>Small container (640px max)</ui-text> </ui-layout-container>
