Skeleton <ui-skeleton> renders animated shimmer placeholders while content is loading. Supports text lines, circles, and rectangles. Basic Usage Use lines for text-shaped skeletons, or variant="circle" / variant="rect" for shapes.
<ui-skeleton lines="3"></ui-skeleton> <ui-skeleton variant="circle"></ui-skeleton> <ui-skeleton variant="rect" width="200px" height="80px"></ui-skeleton>
Variants Three variants for different content shapes.
TEXT (default)
CIRCLE
RECT
<ui-skeleton variant="text" lines="2"></ui-skeleton> <ui-skeleton variant="circle"></ui-skeleton> <ui-skeleton variant="rect" width="150px" height="60px"></ui-skeleton>
Sizes Scale the skeleton with size keywords or any CSS length.
<ui-skeleton size="x-small"></ui-skeleton> <ui-skeleton size="small"></ui-skeleton> <ui-skeleton></ui-skeleton> <ui-skeleton size="large"></ui-skeleton>
Content Composition Combine variants to mimic real layouts — for example, a user card.
<ui-card elevation="1"> <ui-card-body> <ui-skeleton variant="circle" width="3em" height="3em"></ui-skeleton> <ui-skeleton lines="1" width="60%"></ui-skeleton> <ui-skeleton lines="3"></ui-skeleton> </ui-card-body> </ui-card>
