API Reference <ui-card>
AttributeTypeDefaultDescription
backgroundString—Background colour (palette token or CSS colour). Text auto-contrasts.
colorStringautoText colour (auto-contrasts against background if omitted)
sizeString—Card font-size (named keyword or CSS length)
elevationString—Shadow depth: 1–5 (adds border + shadow) or raw CSS box-shadow
outlineBooleanfalseBorder only, no shadow
flatBooleanfalseNo shadow and no border — completely flat surface
clickableBooleanfalsePointer cursor, hover lift effect, fires ui-click event
<ui-card-header>
AttributeTypeDefaultDescription
backgroundStringtransparentHeader background colour (palette token or CSS colour)
colorStringinheritHeader text colour (auto-contrasts against background if omitted)
sizeString—Header font-size (named keyword or CSS length)
borderBooleanfalseShow a bottom border divider between header and body
<ui-card-body>
AttributeTypeDefaultDescription
backgroundStringtransparentBody background colour (palette token or CSS colour)
colorStringinheritBody text colour (auto-contrasts against background if omitted)
sizeString—Body font-size (named keyword or CSS length)
<ui-card-footer>
AttributeTypeDefaultDescription
backgroundStringtransparentFooter background colour (palette token or CSS colour)
colorStringinheritFooter text colour (auto-contrasts against background if omitted)
sizeString—Footer font-size (named keyword or CSS length)
<ui-card-media>
AttributeTypeDefaultDescription
heightStringautoConstrain media height (any CSS length). Images use object-fit: cover.
