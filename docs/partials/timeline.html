Timeline <ui-timeline> renders a vertical event timeline for activity feeds, changelogs, and step-by-step histories. Each event is a <ui-timeline-item> with an optional dot colour and icon. Compose all content inside the default slot using ui-* components. Basic Usage Place <ui-timeline-item> elements inside. Compose headings, timestamps, and body content using ui-text and other components in the default slot.
