Tooltip <ui-tooltip> shows contextual help text on hover or focus. Wrap any element and set the content attribute. Basic Usage Wrap a target element and provide a content string.
Save Delete Badge
<ui-tooltip content="Save your changes"> <ui-button background="indigo-600">Save</ui-button> </ui-tooltip>
Placements Position the tooltip with placement: top (default), bottom, left, or right.
Top Bottom Left Right
<ui-tooltip content="Top" placement="top">...</ui-tooltip> <ui-tooltip content="Bottom" placement="bottom">...</ui-tooltip> <ui-tooltip content="Left" placement="left">...</ui-tooltip> <ui-tooltip content="Right" placement="right">...</ui-tooltip>
Styling Customise colours, size, and max-width.
Indigo Rose Emerald Tiny
<ui-tooltip content="Indigo" background="indigo-600">...</ui-tooltip> <ui-tooltip content="Rose" background="rose-600">...</ui-tooltip> <ui-tooltip content="Small text" size="x-small">...</ui-tooltip>
Disabled Set disabled to suppress the tooltip while keeping it in the DOM.
Disabled tooltip
<ui-tooltip content="Hidden" disabled>...</ui-tooltip>
play-circle Playground Change content, placement, colours, and options interactively.

API Reference
AttributeTypeDefaultDescription
contentString—Tooltip text to display
placementStringtopPosition: top, bottom, left, or right
delayNumber200Show delay in milliseconds
backgroundString—Tooltip background colour (palette token or CSS colour)
colorString—Tooltip text colour
sizeString—Font size: named keyword (xx-small … xxx-large) or CSS length
max-widthString250pxMaximum width of the tooltip popup
disabledBooleanfalseSuppress the tooltip (keeps element in DOM)
