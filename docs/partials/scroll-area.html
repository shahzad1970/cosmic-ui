Scroll Area <ui-scroll-area> provides a styled scrollable container with customisable thin scrollbars that appear on hover, plus subtle scroll-shadow indicators at content edges. Basic Usage Set a fixed height to enable vertical scrolling. Gradient shadows appear automatically at the top and bottom edges to indicate more content.

Scrollbar Styles Choose between thin (hover-reveal), auto (system default), or hidden.
thin (default) Thin scrollbar appears on hover. Scroll down to see it in action with this long content. auto System default scrollbar. Uses the native scrollbar style provided by the browser. hidden Hidden scrollbar. Content is still scrollable but no scrollbar is shown.
<ui-scroll-area height="80px" scrollbar="thin">...</ui-scroll-area> <ui-scroll-area height="80px" scrollbar="auto">...</ui-scroll-area> <ui-scroll-area height="80px" scrollbar="hidden">...</ui-scroll-area>

Scroll End Event The ui-scroll-end event fires once when the user scrolls to the bottom — useful for infinite-scroll or load-more patterns.

API Reference
AttributeTypeDefaultDescription
heightString—Container height (required for vertical scrolling)
max-heightString—Maximum height before scrolling begins
widthString—Container width
sizeString—Component font-size (named keyword or CSS length)
directionStringverticalScroll direction: vertical, horizontal, both
scrollbarStringthinScrollbar style: thin, auto, hidden
paddingString—Inner padding (named size or CSS length)
Methods
MethodDescription
scrollToTop(behavior)Scroll to the top (default: 'smooth')
scrollToBottom(behavior)Scroll to the bottom (default: 'smooth')
scrollToLeft(behavior)Scroll to the left edge (default: 'smooth')
scrollToRight(behavior)Scroll to the right edge (default: 'smooth')
Events
EventDescription
ui-scroll-endFired once when the user scrolls to the bottom. Re-fires if the user scrolls away and back down.
