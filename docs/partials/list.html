List <ui-list> is a styled vertical list with optional dividers, hover states, striped rows, icons, and action slots. Each <ui-list-item> supports leading/trailing slots, descriptions, links, and disabled state. Basic Usage Place <ui-list-item> elements inside. Use dividers and hoverable for visual style.
inbox Inbox 12 star Starred send Sent trash-2 Trash
<ui-list dividers hoverable bordered> <ui-list-item> <ui-icon slot="leading">inbox</ui-icon> Inbox <ui-badge slot="trailing" background="indigo-500" pill>12</ui-badge> </ui-list-item> <ui-list-item description="View sent messages"> <ui-icon slot="leading">send</ui-icon> Sent </ui-list-item> <ui-list-item disabled> <ui-icon slot="leading">trash-2</ui-icon> Trash </ui-list-item> </ui-list>
Striped & Bordered Alternate row backgrounds with striped. Add bordered for an outer border.
folder Documents image Photos music Music film Videos download Downloads
<ui-list striped bordered dividers> <ui-list-item>Documents</ui-list-item> <ui-list-item>Photos</ui-list-item> <ui-list-item>Music</ui-list-item> ... </ui-list>
Link Items Use href on items to render them as clickable links. Add target="_blank" for external links.
github GitHub external-link twitter Twitter external-link
<ui-list dividers bordered> <ui-list-item href="https://github.com" target="_blank"> <ui-icon slot="leading">github</ui-icon> GitHub <ui-icon slot="trailing">external-link</ui-icon> </ui-list-item> </ui-list>
Avatars & Descriptions Combine avatars in the leading slot with description for rich contact-style lists.

API Reference ui-list
AttributeTypeDefaultDescription
sizeString—Component font-size (named keyword or CSS length)
dividersBooleanfalseShow 1 px divider lines between items
hoverableBooleanfalseHighlight items with a subtle background on hover
stripedBooleanfalseAlternate row backgrounds for visual contrast
backgroundString—List background colour — palette token or CSS colour
colorString—List text colour — palette token or CSS colour
borderedBooleanfalseShow a rounded outer border around the list
ui-list-item
AttributeTypeDefaultDescription
valueString—Item value emitted in the ui-click event detail
descriptionString—Secondary text rendered below the main label
disabledBooleanfalseDim the item and prevent interaction
hrefString—Navigate on click — renders the item as an <a> tag
targetString—Link target (_blank, _self, etc.)
Slots
SlotDescription
(default)Main label content of the list item
leadingContent before the label — typically an icon or avatar
trailingContent after the label — typically a badge, button, or icon
Events
EventDetailDescription
ui-click{ value }Fired on ui-list-item when clicked (non-disabled). Detail includes the item's value attribute.
