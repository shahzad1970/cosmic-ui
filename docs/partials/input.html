Input <ui-input> is a rich, accessible text input with built-in labels, help text, error messages, prefix/suffix, clearable, and password-toggle. Basic Usage Use label, placeholder, and prefix / suffix to create rich input fields.
<ui-input label="Email" type="email" placeholder="you@example.com"></ui-input> <ui-input label="Search" type="search" placeholder="Search..." clearable></ui-input> <ui-input label="Amount" prefix="$" suffix="USD" placeholder="0.00"></ui-input>
Help & Error Text Use help for guidance and error for validation messages. When error is set, the input border turns red and the help text is replaced by the error.
<ui-input label="Username" placeholder="Choose a username" help="Must be 3–20 characters"></ui-input> <ui-input label="Username" value="jo" error="Too short — minimum 3 characters"></ui-input>
Password Toggle Set type="password" and toggleable to show a show/hide eye icon.
<ui-input label="Password" type="password" toggleable placeholder="Enter your password"></ui-input>
Required & Readonly required adds an asterisk to the label. readonly prevents editing but keeps the input focusable. disabled dims the entire component and prevents interaction.
<ui-input label="Full Name" required placeholder="Jane Doe"></ui-input> <ui-input label="Account ID" readonly value="ACC-29481"></ui-input> <ui-input label="Archived" disabled value="Locked field"></ui-input>
Sizes The size attribute scales the entire input — label, text, prefix/suffix, and padding.
<ui-input label="Small" size="small" placeholder="Small input"></ui-input> <ui-input label="Default" placeholder="Default input"></ui-input> <ui-input label="Large" size="large" placeholder="Large input"></ui-input>
Background & Colour Set background and color to customise the input wrapper appearance.
<ui-input label="Subtle" background="gray-50" placeholder="Light background"></ui-input> <ui-input label="Branded" background="indigo-50" color="indigo-700" placeholder="Indigo accent"></ui-input>

API Reference
AttributeTypeDefaultDescription
typeStringtextInput type — text, email, password, number, tel, url, search, date, time, datetime-local, month, week, color
valueString—Current input value
placeholderString—Placeholder text shown when empty
nameString—Form field name for submission
labelString—Label text shown above the input
helpString—Help text shown below the input
errorString—Error message — also triggers red border styling
prefixString—Text or symbol before the input (e.g. $)
suffixString—Text or symbol after the input (e.g. USD)
sizeString—Component font-size — named keyword (small, large, …) or CSS length
backgroundString—Background colour — palette token or CSS colour
colorString—Text colour — palette token or CSS colour
disabledBooleanfalseDisables the input (dims, no interaction)
readonlyBooleanfalseMakes the input read-only
requiredBooleanfalseMarks as required — adds * to the label
clearableBooleanfalseShows a clear × button when the input has a value
toggleableBooleanfalseShows a show/hide toggle for password inputs
patternString—Validation regex pattern passed to native input
minlengthNumber—Minimum character length
maxlengthNumber—Maximum character length
minNumber—Minimum value (for number / date types)
maxNumber—Maximum value (for number / date types)
stepNumber—Step increment (for number / date types)
autocompleteString—Browser autocomplete hint (e.g. email, off)
