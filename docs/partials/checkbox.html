Checkbox <ui-input-checkbox> is a labeled checkbox with optional help text, error state, indeterminate mode, and full colour support. Basic Usage Use checked and indeterminate states.
<ui-input-checkbox label="Receive updates" checked></ui-input-checkbox> <ui-input-checkbox label="Partially selected" indeterminate></ui-input-checkbox> <ui-input-checkbox label="Unchecked option"></ui-input-checkbox>
Help & Error Messages Use help for guidance text below the label. Set error to show a red error message (replaces help text).
<ui-input-checkbox label="Subscribe to newsletter" help="We'll send you weekly updates"></ui-input-checkbox> <ui-input-checkbox label="Accept terms" error="You must accept to continue" required></ui-input-checkbox>
Disabled Add disabled to prevent interaction.
<ui-input-checkbox label="Disabled unchecked" disabled></ui-input-checkbox> <ui-input-checkbox label="Disabled checked" disabled checked></ui-input-checkbox>
Sizes Use size with CSS keywords (x-small … xx-large) or any CSS length.
<ui-input-checkbox label="Extra small" size="x-small" checked></ui-input-checkbox> <ui-input-checkbox label="Large" size="large" checked></ui-input-checkbox> <ui-input-checkbox label="Extra large" size="x-large" checked></ui-input-checkbox>
Background & Color Set background to a palette token (e.g. emerald-600) or any CSS colour to change the checked accent.
<ui-input-checkbox label="Emerald" background="emerald-600" checked></ui-input-checkbox> <ui-input-checkbox label="Rose" background="rose-600" checked></ui-input-checkbox>
play-circle Playground Toggle states, size, accent colour, and error text.
