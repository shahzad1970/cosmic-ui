Badge <ui-badge> is a small label for statuses, counts, and tags with shape, colour, and animation controls. Basic Usage Content goes inside the element. Add a background colour for a filled badge.

API Reference
AttributeTypeDefaultDescription
backgroundString—Background colour (palette token or CSS colour). Used for outline border colour too.
colorStringautoText colour (auto-contrasts against background if omitted)
sizeString—Badge size (named keyword or CSS length)
pillBooleanfalseFully rounded capsule shape (border-radius: 9999px)
outlineBooleanfalseBorder-only variant — no fill; text and border use background colour
circleBooleanfalseCircular shape for single-character or icon badges
pulseBooleanfalseGlowing pulse animation radiating outward
bounceBooleanfalseGentle vertical hop animation
shakeBooleanfalseQuick horizontal wiggle animation
pingBooleanfalseExpanding ring beacon animation
