JS-Driven (Sort & Select) Set columns and data via JavaScript for sortable headers and row selection.
const table = document.querySelector('ui-table'); table.columns = [ { key: 'name', label: 'Name', sortable: true }, { key: 'email', label: 'Email' }, { key: 'role', label: 'Role', sortable: true }, ]; table.data = [ { name: 'Alice', email: 'alice@ex.com', role: 'Admin' }, { name: 'Bob', email: 'bob@ex.com', role: 'User' }, { name: 'Charlie', email: 'charlie@ex.com', role: 'Editor' }, ]; table.addEventListener('ui-sort', e => console.log('Sort:', e.detail)); table.addEventListener('ui-select', e => console.log('Select:', e.detail));

API Reference
AttributeTypeDefaultDescription
stripedBooleanfalseAlternate row shading on even rows
borderedBooleanfalseAdd borders around every cell
hoverableBooleanfalseHighlight rows on mouse hover
compactBooleanfalseReduced cell padding for dense data
selectableBooleanfalseShow row checkboxes and enable selection (JS-driven mode)
sortableBooleanfalseEnable click-to-sort on all column headers (JS-driven mode)
sizeString—Table font-size (named keyword or CSS length)
backgroundString—Table background colour (palette token or CSS colour)
sticky-headerBooleanfalseKeep the <thead> pinned while the body scrolls (set a fixed height)
JS Properties
PropertyTypeDescription
columnsArrayColumn definitions: { key, label?, sortable?, align?, render? }
dataArrayArray of row objects — keys must match column key values
selectedRowsArray (getter)Returns the currently selected row objects
Events
EventDetailDescription
ui-sort{ column, direction }Fired when a sortable column header is clicked. direction is "asc" or "desc".
ui-select{ selected, row, index }Fired when row selection changes. row is null for select-all.
